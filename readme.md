# koa-template-app

## example

> 生成字段检测配置文件

```shell
npm i -g typescript-json-schema
yarn validate
```

> 数据库迁移

```shell
yarn migrate mysql
```

> 测试环境运行服务

```shell
yarn dev
```

> 构建依赖

```shell
cd /build/shell
./main.sh
# 等待容器内服务初始化完成后执行
./init.sh
```
